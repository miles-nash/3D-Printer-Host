Welcome to the wonderfull world of 3D Printer Host.
The 3D Printer Host is a hardware/software solution to connect your OctoPrint Enabled 3d printer to a remote monitoring gadget and Amazon Alexa.
For more information about this prooject or to build one for yourself go to: https://www.hackster.io/Milesnash_/3d-printer-monitoring-with-alexa-and-arduino-024292 
This repository serves only to hold the arduino code for the 3D Printer Host.
